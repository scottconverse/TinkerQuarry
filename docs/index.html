TinkerQuarry is built for practical printed parts: clips, mounts, brackets, spacers, trays, organizers, and the one-off fixtures that should not require a full CAD detour.

Editable CAD

Manufacturing gate

Ready to print means the selected printer, material, orientation, and slice are current.

Current product flow

Architecture

The Studio UI runs in a Tauri Windows shell and talks to the local KimCad engine. OpenSCAD renders geometry, PrintProof3D and mesh checks provide readiness evidence, OrcaSlicer produces print output, and trusted CadQuery twins provide STEP where available.

Verified Windows beta

Every release must pass the complete automated test suite, a real build of the Windows installer, and hands-off checks of both the built app and a fresh installed copy before it is published. Malformed or unrecognized 3D files are detected and safely rejected instead of becoming broken designs.

Gate clean

Full repo gate, browser e2e, engine pytest, Rust/Tauri tests, and audit checks passed.

Installer proven

NSIS installer builds from a short Windows path and the installed app smoke passes.

Honest beta limits

Windows-first packaging, advisory visual review, and hardware certification limits stay clearly documented. The installer is signed as of v1.5.0 (SmartScreen may still warn while the new certificate builds reputation).
