3 · Experimental generator — the no-dead-end inline offer (Surface C)

When a request has no template match, the assistant offers the experimental generator rather than dead-ending — with the warning carrying the weight. Never auto-run.

◆
I don't have a precise template for a “topographic coaster of Mt. Rainier.” I can try an experimental generator that writes the shape directly — it isn't always right, it runs in a locked sandbox, and it still has to pass the printability check.
Experimental · may not be perfect. The result could be rough or fail the check — you'll see exactly what happens.
Trust rule 4Offered, never silent; sandboxed; still gated.
